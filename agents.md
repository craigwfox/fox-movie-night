# AGENTS.md

## Setup commands

- Install deps: `pnpm install`
- Start dev server: `pnpm dev`
- Run tests: `pnpm test`

## Tech Stack

- TypeScript
- Next.js 16
- Next.js using SSR
- Supabase for database
- Netlify for hosting

## Code style

- TypeScript strict mode
- single quotes, no semicolons
- Use functional patterns where possible
